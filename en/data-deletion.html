Data Deletion Request

This page provides a clear process for requesting deletion of personal data. It is structured for Meta/Facebook App Review requirements.

1. Submit a Deletion Request

Email bratanbitcoin@gmail.com with the subject line Data Deletion Request.

2. Required Information

  • Platform used (for example Meta/Instagram, YouTube, Whop).
  • Username and/or account ID.
  • If available: additional identifiers (for example Whop, Discord, or YouTube IDs).
  • Optional: short note about affected features or data.

3. Processing Timeline

Requests are acknowledged and typically handled within 30 days, unless legal obligations require longer retention.

4. What Will Be Deleted

  • Internally stored personal data where deletion is legally and technically possible.
  • Temporarily staged upload files.
  • Technical logs/event records, where removal is possible.

5. Third-Party Platform Data

Data already stored directly by platforms (such as Meta/Instagram, YouTube, Whop) may need to be removed separately on those platforms.

6. Self-Service Status

A self-service deletion portal is currently not available. Deletion requests are handled via the email process above.

Deletion contact: bratanbitcoin@gmail.com
